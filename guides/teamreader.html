chaorace's Teamspeak Chat Text To Speech Guide for Windows

Pre-requisites

You must have:

  • A Teamspeak 3 client
  • Must have channel chat logging enabled
  • A Windows computer to run the client on
  • TeamReader by me
  • eSpeak by Jonathan Duddington
  • Basic reading comprehension and patience
  • Don't blame me for stupid shit

Step Zero: Let Go of Your Earthly Bonds

This guide features code (sloppily) written by me! That means you need to acknowledge that I'm not responsible for the hell it may or may not unleash upon the innocent people of the Earth! Once you've confirmed with yourself and your family that you're okay with this, please continue.

Step One: Download and install eSpeak by Jonathan Duddington

Did I mention that eSpeak is made by Jonathan Duddington? Because he's a pretty cool guy for making such a lightweight and flexible TTS program under the GPL and all that. Anyways, you're going to have install his software. TeamReader was designed and tested using version 1.48.04, but feel free to try a newer version if you want. It should still work unless the command-line version of the application is removed or drastically changed. The TeamReader software does, however, expect eSpeak to be installed in the default directory, so please install it there.

Step Two: Download, unpack, and run TeamReader by me

You can unpack TeamReader anywhere you like, but I recommend putting it somewhere where you won't lose it. Once you've finished reading the README.txt file, double-click TeamReader.bat to start the program. It is very important that you do not start the program using TeamReader.jar, because it's much harder to close the program when you're done.

Step Three: Use Teamspeak like you usually would

Now the software should transcribe all chat messages to audio. It even supports multiple servers at once! Simply play like usual and enjoy!
